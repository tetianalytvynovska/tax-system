TaxAgent

Цей проєкт містить:

- Backend 
- Frontend
- Ролі:
  - Адміністратор (2FA по email)
  - Звичайні користувачі (реєструються самостійно)
- Офіційні PDF-декларації для користувачів
- Офіційний агрегований PDF-звіт та CSV (Excel) для адміністратора
- Адмін-дашборд з комбінованими даними (реальні значення з БД + макетні блоки)
- Валідація фільтрів дат (дата "до" не може бути раніше, ніж "з")
- Повідомлення та службові рядки, якщо за вибраний період немає даних
