# TaxAgent – фінальна версія з адмін-дашбордом

Цей проєкт містить:

- Backend (Node.js + Express + SQLite, директорія `backend/`)
- Frontend (React + Vite, директорія `frontend/`)
- Ролі:
  - Адміністратор (email: `tetianalytvynovska@gmail.com`, пароль: `0987654321`, 2FA по email)
  - Звичайні користувачі (реєструються самостійно)
- Офіційні PDF-декларації для користувачів
- Офіційний агрегований PDF-звіт та CSV (Excel) для адміністратора
- Адмін-дашборд з комбінованими даними (реальні значення з БД + макетні блоки)
- Валідація фільтрів дат (дата "до" не може бути раніше, ніж "з")
- Повідомлення та службові рядки, якщо за вибраний період немає даних

## Запуск

1. Встановити залежності в корені:

   ```bash
   npm install
   ```

2. Налаштувати SMTP:

   - Скопіювати `backend/config.example.js` у `backend/config.js`
   - Вказати дані Gmail (або іншого SMTP) та app password

3. Запустити backend:

   ```bash
   npm run dev:backend
   ```

4. В іншому терміналі запустити frontend:

   ```bash
   cd frontend
   npm install
   npm run dev
   ```

5. Відкрити у браузері:

   ```
   http://localhost:5173
   ```
